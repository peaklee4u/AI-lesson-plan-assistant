# Jigsaw I 모형 (과제분담 협동학습 I)

## 1. 정의

Jigsaw I 모형은 수업 방식이 퍼즐 조각을 맞추는 것과 유사하다고 하여 붙여진 이름의 협동학습 모형이다. 원래는 학업 성취보다 인종 간·문화 간의 교우 관계와 같은 정의적 측면의 향상이 일차적 목표였으나, 현재는 구성원의 상호의존적 활동을 요구하는 과제 지향적 협동학습 모형으로 주로 사용되고 있다. 전체 학습 과제를 모둠 구성원 수에 맞게 분할하여 각 학생이 서로 다른 부분을 담당하고, 같은 부분을 맡은 학생들이 전문가 모둠에서 학습한 후 원 모둠으로 돌아가 동료들에게 가르치는 구조를 가진다.

## 2. 이론적 배경 및 특징

**주요 특징:**
- **높은 자원 상호의존성**: 학습 과제를 성공적으로 완성하기 위해서는 각 구성원이 다른 구성원들의 자원(정보)이 필요하므로 긍정적 자원 상호의존성이 자연스럽게 형성됨
- **낮은 보상 상호의존성**: 평가에서 개인 점수만 사용하고 집단 점수를 반영하지 않기 때문에 과제 해결의 상호의존성은 높으나 보상의 상호의존성은 낮음
- **전문가-교사 경험**: 전문가 모둠에서 특정 부분에 대한 전문성을 기르고, 원 소속 모둠에서 동료에게 가르치는 이중적 학습 경험 제공
- **과제 분담을 통한 협동**: 보상 구조가 아니라 학습 과제의 분담을 통해서 모둠 구성원들이 서로 의존하고 협동하도록 하는 구조적 특성
- **개념 학습에 효과적**: 과학학습에서 탐구 기능 신장보다는 개념 획득에 목적을 둔 교수학습에 더 효과적

## 3. 단계별 활동

### **1단계: 모둠 구성**
서로 이질적인 5~6명의 학생으로 모둠을 구성한다. 성별, 학업 능력, 가능한 경우 인종, 종교, 사회적 배경도 고려하여 다양한 집단이 형성되도록 한다. 이질적 구성일수록 Jigsaw의 효과가 높아진다.

### **2단계: 개인별 전문 과제 부과**
전체 학습 내용을 모둠 구성원의 수와 동일한 수로 나누어 각 소주제의 내용을 전문 과제로 각각의 학생에게 부과한다. 각 학생은 자신에게 부과된 소주제만을 담당하며, 다른 소주제는 접하지 않는다.

**예시 (소화기관계의 구조와 기능, 5명 구성):**
- 학생 A: 입(구강)의 구조와 기능
- 학생 B: 위의 구조와 기능  
- 학생 C: 소장의 구조와 기능
- 학생 D: 대장의 구조와 기능
- 학생 E: 간의 구조와 기능

### **3단계: 전문가 모둠에서의 협동 학습**
전문 과제를 하나씩 맡은 학생들은 각자 원 소속 모둠에서 나와 전문 과제의 주제가 같은 다른 모둠의 구성원들과 함께 전문가 모둠을 형성한다. 전문가 모둠에서 학생들은 과제 해결을 위해 협동 학습을 하며, 담당 소주제에 대해 깊이 있게 토의하고 학습한다.

**전문가 모둠에서의 활동:**
- 담당 소주제에 대한 자료 분석 및 토의
- 핵심 개념 정리 및 설명 방법 협의  
- 원 모둠에서 동료들에게 가르칠 내용과 방법 준비

### **4단계: 원 소속 모둠에서의 협동 학습**
전문가 모둠에서 협동 학습이 끝나면 자신의 원 소속 모둠으로 돌아가 동료들에게 전문가 모둠에서 학습한 내용을 가르치고 설명한다. 각 학생이 동료 교수자 역할을 수행하며, 모든 구성원이 퍼즐의 모든 조각을 맞추듯 전체 학습 내용을 완성하게 된다.

### **5단계: 개별 평가**
학생들은 전체 학습 과제에 대해 개인별 평가를 받는다. 자신이 직접 학습한 소주제뿐만 아니라 동료에게 배운 내용 전체에 대해 평가받는다.

### **6단계: 개인별 점수 산출**
각 학생의 개인별 점수만 계산될 뿐 향상 점수나 모둠 점수는 계산하지 않고 모둠 보상 또한 없다.

## 4. 적용 시 고려사항

**① 분담 가능한 과제 구조**
과제 내용이 분담 학습이 가능하도록 분절될 수 있어야 한다. 분담한 과제가 서로 관련성이 높고 위계적이거나 인과적인 관계에 있는 내용이라면, 학습의 순서가 명확한 내용이므로 뒤에 배워야 할 내용을 담당한 학생들이 학습 내용 이해에 어려움이 생길 것이다. 따라서 학습 과제는 서로 독립적이고 분절적인 내용으로 구성된 것이어야 한다.

**② 적절한 학습 내용 수준**
학습 내용의 수준이 학생들의 학습 능력에 적절해야 한다. 학습 내용의 수준이 하 수준 학생들도 무난히 학습할 수 있는 정도에서 맞추어지는 것이 중요하다. 학습 수준이 낮은 학생들이 자기에게 주어진 과제를 잘 수행하지 못할 경우에 같은 모둠에서 학습하고자 하는 다른 학생들에게도 부정적인 영향을 미칠 수 있다.

**③ 효율적인 시간 관리**
시간을 비효율적으로 활용하여 학습이 충분히 이루어지지 않을 수도 있고, 학생들이 무질서하게 행동하여 학습 분위기를 망칠 가능성도 있다. 따라서 교사의 철저한 안내와 통제가 필요하다.

**④ 다른 학습 모형과의 병행**
Jigsaw I 모형이 가지고 있는 단점을 최소화하기 위해 다른 학습 모형과 병행하여 활용함으로써 보완하는 것이 바람직하다.

### **(2) 과제 설계**
- [ ] 전체 학습 내용이 구성원 수에 맞게 균등하게 분할되었는가?
- [ ] 각 소주제가 서로 독립적이고 분절 가능한 내용으로 구성되었는가?
- [ ] 위계적·인과적 순서가 강한 내용이 과제로 선정되지 않았는가?
- [ ] 학습 내용의 수준이 하 수준 학생도 수행 가능한 정도인가?

### **(3) 전문가 모둠 활동**
- [ ] 동일 소주제 담당 학생들이 전문가 모둠을 형성하고 충분히 학습하는가?
- [ ] 전문가 모둠에서 핵심 개념 정리와 설명 방법 협의가 이루어지는가?
- [ ] 원 모둠에서 가르칠 내용을 준비하는 시간이 충분히 확보되는가?

### **(4) 원 모둠 협동 학습**
- [ ] 모든 학생이 동료 교수자 역할을 충실히 수행하도록 설계되었는가?
- [ ] 각 학생의 설명을 통해 모든 구성원이 전체 내용을 이해할 수 있는 구조인가?
- [ ] 학습 내용의 이해도를 확인하는 상호 점검 활동이 포함되어 있는가?

- [ ] 개인별 평가가 전체 학습 내용을 대상으로 이루어지는가?
- [ ] 자신의 담당 소주제뿐 아니라 동료에게 배운 내용도 평가 범위에 포함되는가?
- [ ] 특정 소주제 담당 학생에게만 유리/불리하지 않도록 평가가 설계되었는가?

### **(6) 전체 수업 설계**
- [ ] 과제 분담을 통한 자원 상호의존성이 자연스럽게 형성되도록 설계되었는가?
- [ ] 개념 학습 중심의 주제에 Jigsaw I이 적용되었는가?
