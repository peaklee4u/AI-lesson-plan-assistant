# 3단계 순환학습 모형 (3-Stage Learning Cycle)

## 1. 정의

3단계 순환학습 모형은 Robert Karplus와 Chester Lawson에 의해 개발된 교수학습 모형으로, 학습자가 구체적인 사물이나 현상을 관찰하고 그들 사이에 존재하는 규칙성을 귀납적으로 찾아내어 스스로 개념이나 원리를 발견하도록 하는 경험주의적 교육 방법론이다. '탐색(Exploration)', '용어(개념) 도입(Term/Concept Introduction)', '개념 적용(Concept Application)'의 3단계를 순환하며, Piaget의 인지발달 이론에 기반하여 학습자의 인지적 갈등과 평형화 과정을 통해 과학적 개념을 구성하도록 돕는다.

## 2. 이론적 배경 및 특징

**역사적 발전과정:**
1957년 Karplus가 딸의 초등학교 수업에서 과학교육 프로그램을 구상하기 시작하여, 1961년 Piaget 연구소 방문을 통해 인지발달 이론을 접목하였다. 1967년 Karplus와 Herbert Thier가 '탐색', '발명', '발견'의 3단계를 명확히 정의했으며, 1977년 교사들의 혼동을 줄이기 위해 '탐색', '개념 도입', '개념 적용'으로 용어를 수정하였다. 1980년대 Lawson은 두 번째 단계를 '용어 도입'으로 재수정하여, 개념은 학생들이 스스로 만드는 것이며 교사는 용어를 소개하는 역할임을 강조하였다.

**주요 특징:**
- **인지적 갈등과 평형화**: 새로운 경험이 비평형 상태를 야기하고, 자기 조절 메커니즘을 통해 평형화를 추구
- **경험 선행 원칙**: 과학적 용어나 개념 도입 이전에 구체적인 탐색 경험을 먼저 제공
- **귀납적 사고 강조**: 구체적 사례에서 일반적 원리나 개념을 도출하는 사고 과정 중시
- **학습자 주도성**: 탐색 단계에서 학생의 자율적 탐구 활동을 최대한 보장

## 3. 단계별 활동

### **1단계: 탐색 (Exploration)**
학생들이 교사의 최소한의 안내 하에 새로운 개념과 현상을 스스로 탐구하는 단계이다. 자연현상이나 학습 자료를 관찰, 분류, 실험하며 데이터를 수집하고, 기존 지식과 새로운 정보 간의 인지적 갈등을 경험한다. 이 과정에서 학생들은 서로의 의견을 교환하고 토론하며, 발견한 내용을 자신의 언어로 기술하고 표현한다.

**교사의 역할:**
- 인지적 갈등을 유도할 수 있는 자료나 학습 상황 제공
- 학생들의 자율적 탐색 활동과 사회적 상호작용 촉진
- 개별 접촉을 통한 이해 수준 파악 및 방향 안내
- 허용적이고 자유로운 학습 분위기 조성

### **2단계: 용어(개념) 도입 (Term/Concept Introduction)**
탐색 단계에서 학생들이 발견한 규칙성과 인과적 설명을 과학적 개념 및 용어와 연결하는 단계이다. 학생들은 탐색 결과를 자신의 언어로 정리하고 발표하며, 교사는 이를 과학적 용어와 연결하여 새로운 개념을 도입한다. 과학적 개념은 학생들이 스스로 발명하기 어렵기 때문에 교사의 적절한 개입이 필요하다.

**핵심 원칙:**
- 학생의 탐색 경험을 바탕으로 한 개념 도입
- 학생의 일상 언어와 과학 용어 간의 연결
- 단순한 정의 전달이 아닌 조작 활동을 통한 이해 촉진
- 인지적 비평형에서 평형 상태로의 전환 지원

### **3단계: 개념 적용 (Concept Application)**
학습한 과학적 개념을 다양한 새로운 상황과 현상에 적용하여 개념을 인지 구조에 정착시키는 단계이다. 새로운 상황에 개념을 적용함으로써 개념의 유용성을 실감하고, 특정 상황에 제한되지 않도록 개념의 추상화와 일반화를 이룬다.

**교사의 역할:**
- 다양한 상황에서의 개념 적용 기회 제공
- 개념의 일반화와 전이 촉진
- 적용 경험 부족으로 인한 제한적 이해 방지

## 4. AI 피드백용 체크리스트

### **(1) 탐색 단계**
- [ ] 학생들의 인지적 갈등을 유발할 수 있는 적절한 현상이나 자료가 제시되었는가?
- [ ] 교사의 직접적 설명 이전에 학생 주도의 탐구 활동이 충분히 보장되는가?
- [ ] 학생들이 자신의 언어로 관찰 결과를 표현하고 동료와 공유하는 기회가 있는가?
- [ ] 사회적 상호작용을 통한 다양한 견해 인식이 촉진되는가?

### **(2) 용어(개념) 도입 단계**
- [ ] 학생의 탐색 결과를 먼저 수렴한 후 과학적 개념을 도입하는가?
- [ ] 학생의 일상 언어와 과학 용어 간의 연결이 명확히 이루어지는가?
- [ ] 개념이 단순 정의가 아닌 조작 활동과 함께 도입되는가?
- [ ] 탐색 경험이 충분하지 않은 상태에서 성급한 개념 도입이 이루어지지 않는가?

### **(3) 개념 적용 단계**
- [ ] 학습한 개념을 새롭고 다양한 상황에 적용하는 활동이 포함되어 있는가?
- [ ] 개념의 일반화와 추상화를 촉진하는 과제가 제시되는가?
- [ ] 단순 반복이 아닌 개념의 폭을 확장하는 방향으로 적용 활동이 구성되었는가?

### **(4) 전체 수업 설계**
- [ ] 탐색 → 개념 도입 → 개념 적용의 순환 구조가 명확히 유지되는가?
- [ ] 각 단계별 시간 배분이 적절하고, 인지적 평형화 과정을 충분히 지원하는가?
- [ ] 학생의 능동적 참여와 교사의 적절한 역할 전환이 설계되어 있는가?
