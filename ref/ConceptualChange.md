# 개념변화 학습 모형 (Conceptual Change Model)

## 1. 정의

개념변화 학습 모형은 학습 전후의 인지구조 변화와 선행 개념의 중요성을 강조하는 교수학습 모형이다. 학생이 이미 가지고 있는 대안적 개념틀(오개념)을 출발점으로 삼아, 과학적으로 더 적합한 개념으로 인지 구조를 재구성하도록 돕는다. Driver, Posner, DiSessa 등이 대표적인 학자로, 학생들이 기존 개념을 인식하고 인지적 갈등을 경험하며 새로운 과학적 개념을 능동적으로 구성하도록 유도한다.

## 2. 이론적 배경 및 특징

**이론적 배경:**
- **Piaget의 인지발달 이론**: 아동의 인지 구조가 학습 과정에서 동화(assimilation)와 조절(accommodation)을 통해 인지적 평형을 이루어간다고 설명
- **Ausubel의 유의미 학습 이론**: 학습자가 새로운 개념을 효과적으로 이해하기 위해서는 선행 개념과의 연관성을 통해 의미 있게 학습해야 한다고 주장
- **과학철학적 배경**: Kuhn의 과학혁명 이론과 Lakatos의 과학적 연구 프로그램에서 중심 약속의 변화에 따른 패러다임 전환 개념을 교육에 적용

**주요 특징:**
- **선개념·대안 개념의 중요성 강조**: 새로운 개념 이해는 기존 개념과의 관계 속에서 이루어진다고 봄
- **인지 갈등 기반의 개념 재구성**: 상충되는 상황을 경험하며 기존 개념에 대한 '불만족'을 느끼고 개념을 변화시킴
- **과정으로서의 변화**: 생각 표현 → 상충 경험 → 새로운 생각 구성·평가 → 적용·성찰의 순환
- **구성주의적 관점**: 개념은 외부에서 주입되는 것이 아니라 학습자가 스스로 재구성한다고 봄

## 3. 단계별 활동 (Driver의 CLIS 모형)

### **1단계: 생각의 표현**
학생들이 학습 주제나 현상에 대해 자신이 가지고 있는 생각을 스스로 표현하도록 유도하는 과정이다. 교사는 학습 주제와 관련된 현상·질문·시범을 제시하고, 학생들은 이를 관찰하고 자신의 생각을 설명하며 기존 지식에 기반한 인식을 표현한다. 이 단계의 목표는 학생들이 가지고 있는 기존 개념을 명확히 드러내고, 개념 변화를 위한 기초를 마련하는 것이다.

### **2단계: 생각의 재구성**

**① 명료화와 교환**
학생들이 자신의 생각을 발표하여 공유하는 시간이다. 다른 학생의 의견과 자신의 의견을 비교하며 의미를 명료화하고, 서로의 장단점을 비교할 기회를 갖는다.

**② 상충된 상황에 노출**
교사는 학생들의 생각과 상충되는 현상이나 사건을 제시한다. 상충된 상황의 노출로 학생은 자신이 알고 있는 개념이 불충분하다는 것을 인식(불만족)함으로써 자신의 개념을 바꾸고자 하는 열린 생각을 갖게 된다.

**③ 새로운 생각의 구성**
학생은 자신의 부족한 생각을 대체할 수 있는 새로운 생각을 구성한다. 학생들이 스스로 구성하기 어려운 과학 개념은 교사가 제시해주며, 이때 제시되는 과학개념은 학생들이 쉽게 이해할 수 있어야 하고 동료 학생들의 생각과 비교해 장단점이 잘 드러나야 한다.

**④ 새로운 생각의 평가**
다른 학생들에게 발표하고 동료 학생의 평가를 통하여 자신이 가지고 있는 개념의 타당성을 평가하도록 하는 단계이다.

### **3단계: 생각의 응용**
학생들이 새롭게 학습한 개념을 실제 상황에 적용해보는 과정이다. 학생들이 습득한 과학 개념이 새로운 소재나 상황에 얼마나 잘 적용될 수 있는지 경험하며, 학습한 개념을 다양한 문제에 적용해보는 기회를 갖는다.

### **4단계: 생각의 변화 검토**
학생들이 자신의 개념이 학습 전후로 어떻게 변화하였는지 평가하는 과정이다. 학습하기 전 가지고 있던 기존 개념과 새롭게 학습한 개념을 비교하며, 개념 변화의 정도와 그 차이점을 인식한다. 이를 통해 학생들은 자신의 학습 과정을 성찰하고, 자신의 인지적 변화를 확인하며 학습 결과를 스스로 평가하는 능력을 기른다.

## 4. AI 피드백용 체크리스트

### **(1) 선개념 파악**
- [ ] 수업 초기에 학생의 기존 생각을 드러내는 활동(발문, 글쓰기, 그림 등)이 있는가?
- [ ] 다양한 대안 개념이 안전하게 표현될 수 있는 분위기가 조성되었는가?

### **(2) 상충된 상황 설계**
- [ ] 학생의 기존 개념으로는 설명하기 어려운 현상이 제시되었는가?
- [ ] 상충 상황이 너무 복잡하거나 애매하지 않고, 핵심 갈등이 분명한가?
- [ ] 학생이 실제로 "이상하다, 내 생각과 다르다"는 불만족을 경험하도록 설계되었는가?

### **(3) 새로운 개념 제시·구성**
- [ ] 새로운 과학 개념은 학생 수준에 맞게 이해 가능한 형태로 제시되는가?
- [ ] 학생이 기존 개념과 새 개념을 비교·대조해 볼 기회가 제공되는가?
- [ ] 교사가 곧바로 '정답'을 선언하기보다, 학생의 재구성 과정을 존중하는가?

### **(4) 개념의 평가·응용**
- [ ] 학생이 새로운 개념을 사용하여 다양한 예·문제를 설명·예측해 보는가?
- [ ] 새 개념이 기존 개념보다 설명력·예측력이 높다는 경험을 하도록 설계되었는가?

### **(5) 생각의 변화 성찰**
- [ ] 수업 전·후 생각을 비교하는 글쓰기, 그림, 토의 등이 포함되어 있는가?
- [ ] 학생이 자신의 개념 변화 과정을 언어화·반성해 보도록 요구하는가?

### **(6) 전체 수업 설계**
- [ ] 수업의 초점이 지식 전달이 아니라 개념 재구성 과정에 맞춰져 있는가?
- [ ] 개념 변화를 이끌기 위한 시간과 상호작용(토의·질문)이 충분한가?
- [ ] 대안 개념을 단순히 '틀린 것'으로 낙인찍지 않고, 변화의 발판으로 활용하는가?
