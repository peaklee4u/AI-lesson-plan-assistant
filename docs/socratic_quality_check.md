# 소크라테스 응답 질적 검증 가이드라인

본 가이드는 연구자가 AI Assistant의 4단계 응답 샘플(50개 권장)을 검토할 때 준수해야 할 평가 기준입니다.

## 1. 정답 미제공 원칙 (Strict Non-Solution)
- [ ] AI가 직접적인 수정안(Rewrite)을 제시하지 않았는가?
- [ ] "추가하세요", "바꾸세요" 등 명령형으로 지시하지 않았는가?
- [ ] 학생이 생각할 시간을 뺏는 정보 과잉(Information Overload)은 없는가?

## 2. 질문의 교육학적 적절성
- [ ] 질문이 단순히 "왜?"가 아니라, 구체적인 교수학습이론(예: 5E의 탐색 단계)을 근거로 하는가?
- [ ] 질문이 학습자의 기존 지도안 텍스트를 구체적으로 인용하거나 참조하고 있는가?
- [ ] 질문이 상위 인지(Metacognition)를 자극하여 학생이 자신의 설계 의도를 성찰하게 하는가?

## 3. 대화의 연속성 및 쟁점 유지
- [ ] 한 번의 응답에 하나의 쟁점만 다루고 있는가?
- [ ] 학생의 답변 내용을 무시하지 않고 수용(Acknowledgment)한 뒤 후속 질문을 하는가?
- [ ] 3-5턴의 대화가 유지된 후 다음 쟁점으로 논리적으로 전환되는가?

## 4. 거부 사례 (Rejection Cases)
다음과 같은 응답은 연구 데이터로서 부적합하거나 시스템 수정이 필요합니다.
- 학생의 학문적 오류를 정정해주지 않고 무조건 동의하는 경우
- 루브릭과 무관한 일반적인 칭찬만 반복하는 경우
- 소크라테스식 질문이 아닌 단순한 "예/아니오" 단답형 질문
